# Active Plans

Use this directory for work that is in progress or ready to implement.

## Plan Template

```markdown
# Plan title

## Goal

Concrete outcome.

## Tasks

- [ ] Task 1.
- [ ] Task 2.
- [ ] Verify.

## Decisions

- Decision made during implementation.

## Verification

- Command/result.
```

Move completed plans to `docs/plans/completed/`.
